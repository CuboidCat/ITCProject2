Project 1: Linux

Linux is an example of an open source operating system that, over the years, has been refined and developed constantly in order to provide a secure alternate operating system that goes outside of what Windows and Mac offer.

As an operating system, Linux's popularity stems from it being a more customizable operating system compared to both Windows and Mac. Since it is an open source project, the operating system can be refined and modified to multiple extents. For example, the Android operating system used in mobile phones is, essentially, a modified version of Linux.
